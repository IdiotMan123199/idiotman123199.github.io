Nostalgic Weaponry
Nostalgic Weaponry is one of my first major game projects I've worked on.
It is a hero-based shooter game inspired by tools made by ROBLOX during 2006.
The game features a huge array of weapons, a testing area, and two separate versions: original and remastered.
One unique weapon is the Ban Hammer, which allows players to send the nearest opponent into a replica of the map for a 1v1 duel.
Nostalgic Weaponry Classic
Nostalgic Weaponry Classic is the first version of Nostalgic Weaponry.
This is similar to the remastered version, but differs slightly.
Similar weapons are inside but with different abilities.
It may be a little janky and laggy as the code is not as good in this.
Obby Master
Obby Master is an unofficial recreation of game called "Obby Leader", which has been recognized and appreciated by the original
creator,
it was created within a month or so. The game starts with a selection of a "Master" with their mission to build a obby (parkour)
that is hard enough that the rest cannot beat. The game has features like:
pre-made traps which can be placed, cosmetics for masters, boosts non-masters can use, pro servers, etc etc.

Musical Shooting
Musical Shooting is my first game playable on itch.io and made with Unity.
It was part of a very small game jam called Rage Game Jam, with the theme: "Something needs to be charged" and placed #5 out of #5 overall on the game jam.
The game is a game where you must shoot music boxes and you are constantly spinning. For every shot,
you switch directions. There are 5 levels and a combo bar that charges up slowly to activate a unique ability.
Nothing shall be found...
The second playable game I've made on Unity for itch.io and it's probably my best one, this was made for Abyssal's Pixel Game Jam #7 with the theme: "No one sees you".
The game follows 'Agent Angus' whose duty is to infilitrate the building and return with the diamond!
It takes inspiration from games like Payday 2 and Entry Point from ROBLOX.
There are 7 levels and 7 challenge levels which buff the difficulty of the original by alot.
Calucational Integers
Calucational Integers is my first game made in Godot as part of the Jump Ship Jam with the theme: "Strength in Numbers", placing 37/70 overall and 12th on Theme Interpertation.
It is a platformer puzzle/shooter hybrid game where you have to beat the levels using mathematical functions.
You play as the variable, x, and have a complex value of ai + b. You can shoot your function gun to move objects, attack enemies, etc, however it subtracts your value.
Pressing space also consumes your imaginary value, i.
The Tape
The Tape is my third game made using Unity and was for the GTMK Game Jam 2024 with the theme: "Built to Scale".
